mecab-python 0.98pre3 の Windows 用バイナリ

Windows XP にて MeCab 0.98pre3 の Python バインディングをコンパイルしてみました。

  • libmecab-1.dll
  • mecab-python Python 2.5.4 用
  • mecab-python Python 2.6.2 用
  • IPA辞書(UTF-8)

libmecab.dll, mecab-python, IPA 辞書の3つと mecabrc を組にして使ってください。 mecabrc は空のファイルでかまいませんが、辞書を指定しておくと便利です。

dicdir = ipadic

以下は簡単な使用例です。

>>> import MeCab
>>> s = u'すもももももももものうち'
>>> tagger = MeCab.Tagger('-r mecabrc')
>>> result = tagger.parse(s.encode('utf-8'))
>>> result = result.decode('utf-8')
>>> print result
